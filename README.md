## 📌 About This Repository

This repository contains **40 hands-on practice projects** built while learning Power BI through the  
**30 Days Power BI Micro Course** by **Sathish Dhawale**, Founder of [SkillCourse](https://skillcourse.in).

Each folder covers a specific chart type, visual, or DAX concept — progressing from basic charts all the way to advanced DAX formulas and dashboard design.

---

## 📁 Repository Structure

Every folder follows this structure:

```
Folder_Name/
│── dataset.xlsx          ← Raw data used in the dashboard
│── dashboard.pbix        ← Power BI report file
```

## 🚀 How to Use

1. Clone or download this repository
2. Open any folder of your choice
3. Open the `.pbix` file in **Power BI Desktop**
4. Explore the visuals, DAX measures, and data model
5. Modify the dataset and experiment on your own

```bash
git clone https://github.com/DHAMMADEEPRAMTEKE30/powerbi-practice-files.git
```
