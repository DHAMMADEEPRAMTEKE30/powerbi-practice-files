# 📊 Power BI Practice Dashboards

This repository contains a collection of **Power BI practice projects** created to master different types of visualizations, DAX formulas, and dashboarding techniques.

Each folder represents a specific chart or feature in Power BI and includes:

* 📁 Dataset (Excel file)
* 📊 Power BI Dashboard (.pbix file)

---

## 📂 Project Structure

Each folder follows this structure:

```
Chart_Name/
│── data.xlsx
│── dashboard.pbix
```
